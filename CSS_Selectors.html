CSS selectors(選擇器)

CSS selectors(選擇器)是用來選取要套用CSS樣式的HTML元素的一種語法規則
以下是比較常見的幾種CSS Selectors(選擇器)


  • Universal selectors(通用選擇器）
  • Type selector(標籤選擇器)
  • Class selector(類別選擇器)
  • ID selector (ID選擇器)
  • Attribute selectors(屬性選擇器)
  • Grouping selectors(分組選擇器)
  • Descendant selectors(後裔選擇器)

Universal selectors(通用選擇器）

套用在全部的元素裡

 
        // *代表全部，因此style會套用在全部的元素中

        * {
         color: green; 
        }

Type selector(標籤選擇器)

套用在指定的元素

 
          /* 全部的<a>元素。*/
            a {
              color: red;
            }

Class selector(類別選擇器)

用以選擇所有符合指定class屬性值的元素

 
         /* 全部元素有 class="spacious" */
        .spacious {
          margin: 2em;
        }

ID selector (ID選擇器)

用以選擇指定id屬性值的元素。（一個文件中，每個 ID 屬性都是唯一的。)

 
        /* 元素有 id="demo" */
        #demo {
            border: red 2px solid;
          }

Attribute selectors(屬性選擇器）

 
        /*只要是input裡type是text的字就會變成紅色*/
        input[type="text]{
            color:red;
            }

Grouping selectors(分組選擇器)

不同元素可以用,隔開，例如：

 
        h1, h2, h3, h4, h5, h6 { 
            font-family: helvetica; 
        }
      
 
        div,
        nav {
        font-size: 20px;
        }

Descendant selectors(後裔選擇器）

如果你的網頁裡面有二個以上的<a>(假設分別在div、li裡）若只要對於li裡的<a>元素設定背景顏色，則可以寫成 

 
        li a {
            background-color: aqua;
          }
